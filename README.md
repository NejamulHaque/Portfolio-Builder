# 🛠️ AI Portfolio Builder

Create your own professional portfolio website in minutes using this **AI-powered Portfolio Builder**. Just fill in your information, and the app generates a live, responsive, and elegant portfolio you can preview, export, or host!

---

## 🚀 Features

- ✨ Dynamic form-based portfolio generation
- 🖼️ Live real-time preview while typing
- 📷 Profile photo upload (Base64 or URL)
- 🔗 Social media icons (LinkedIn, GitHub, Instagram, Portfolio)
- 🎨 Color themes and background styles
- ✅ Floating labels and form validation
- 💾 Export portfolio to **PDF**
- 🔗 Shareable link (optional)
- 📱 Fully responsive on all devices

---

## 📦 Tech Stack

- **Frontend**: React.js + Tailwind CSS
- **Backend**: Node.js + Express (optional for data saving)
- **PDF Export**: html2pdf.js
- **Styling**: Tailwind CSS, Framer Motion (for animations)
- **Database (Optional)**: MySQL / MongoDB

## 🧪 Getting Started

1. **Clone the project:**

   ```bash
   git clone https://github.com/yourusername/portfolio-builder.git
   cd portfolio-builder

2.	Install frontend dependencies:
    ```bash
    cd client
    npm install
    npm start

3.	(Optional) Set up backend server:
    ```bash
    cd server
    npm install
    npm start
